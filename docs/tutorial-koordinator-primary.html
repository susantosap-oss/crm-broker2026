Panduan lengkap untuk Koordinator dalam mengelola proyek developer — dari input hingga menerima leads dari website mansionpro.id.

1

Login CRM

Masuk ke sistem CRM Mansion Realty
1

Buka CRM di browser atau PWA

Akses crm.mansionpro.id di Chrome/Safari. Rekomendasi: install sebagai PWA untuk pengalaman terbaik.

2

Masukkan Email & Password

Gunakan email yang sudah didaftarkan Admin. Jika lupa password, minta Admin untuk reset.

3

Klik Masuk → Dashboard tampil

Setelah login, Anda langsung masuk ke Dashboard. Di pojok kiri bawah tampil nama dan badge role Koordinator.

2

Input Project Baru

Ajukan proyek developer untuk ditampilkan di website
1

Buka menu Primary

Tap ikon ⭐ Primary di navigasi bawah CRM. Ini adalah halaman khusus proyek developer.

2

Tap tombol + Tambah Proyek

Tombol ada di pojok kanan atas halaman Primary. Form input akan muncul.

3

Isi semua field wajib

Field bertanda bintang (*) wajib diisi. Semakin lengkap data proyek, semakin menarik tampilannya di website.

4

Tap Ajukan Proyek

Proyek tersimpan dengan status ⏳ Pending dan menunggu review dari Principal.

⏳

Pending — Menunggu Review Principal

Proyek sudah masuk sistem tapi belum disetujui. Principal akan mereview kelengkapan data. Anda bisa edit proyek selama status masih Pending.

✅

Aktif — Disetujui Principal

Proyek sudah disetujui namun belum tayang di website. Admin akan me-review dan mengubah ke Publish. Anda mendapat notifikasi saat status berubah.

🌐

Publish — Tayang di Website ✨

Proyek sudah tampil di mansionpro.id/projects dan dapat ditemukan calon pembeli. Anda mulai bisa menerima leads dari website.

⭐
Pantau status di Dashboard: Buka Dashboard → panel "Daftar Primary Aktif" → lihat jumlah Pending, Aktif, dan Publish proyek Anda secara real-time.
ℹ️
Koordinator tidak bisa mengubah status secara langsung. Status Publish/Draft hanya bisa diubah oleh Admin. Jika perlu percepatan, koordinasi dengan Principal / Admin kantor.
4

Edit Project

Hanya bisa mengedit proyek yang Anda ajukan sendiri
🔒
Aturan Edit: Anda hanya dapat mengedit proyek di mana Anda terdaftar sebagai Koordinator. Proyek milik koordinator lain tidak bisa diedit.
1

Buka menu Primary → tab "Milik Saya"

Tab ini menampilkan semua proyek yang Anda koordinasi, termasuk yang masih Pending.

2

Tap kartu proyek → tap Edit

Tombol Edit hanya muncul untuk proyek milik Anda. Proyek milik koordinator lain hanya bisa dilihat.

3

Update data → tap Simpan Perubahan

Semua field bisa diubah kecuali Koordinator ID (yang ditetapkan sistem) dan Status Publish (hak Admin).

7

Leads dari Website

Privilege Koordinator: terima prospek langsung dari mansionpro.id
⭐
Keunggulan Koordinator Primary: Setiap pengunjung yang klik tombol "Hubungi via WA" atau mengisi form kontak di halaman proyek Anda di website, leads-nya otomatis masuk ke CRM Anda.
🔄 Alur Masuk Leads dari Website
🌐

Pengunjung buka mansionpro.id/projects

Calon pembeli menemukan proyek Anda via Google, media sosial, atau link yang Anda share.

🏠

Pengunjung buka halaman detail proyek

Halaman menampilkan info proyek, foto, spesifikasi, dan tombol kontak. Nama Anda tampil sebagai agen proyek.

📋

Pengunjung klik "Hubungi via WA" / isi form

Sistem meminta nama & nomor WA pengunjung sebelum membuka WhatsApp ke nomor Anda.

💾

Lead otomatis tersimpan di CRM

Data pengunjung (nama, nomor WA) langsung masuk ke menu Leads di CRM Anda dengan sumber "Website".

🔔

Notifikasi masuk ke CRM & Telegram

Anda mendapat notifikasi in-app dan (jika Telegram sudah dihubungkan) pesan Telegram berisi data lead baru.

💬

Follow-up lead secepat mungkin

Buka CRM → Leads → tap lead baru → klik WA untuk langsung hubungi. Lead web yang direspons dalam 5 menit pertama 8x lebih mungkin convert!

💡 Tips Maksimalkan Leads dari Website
📸

Lengkapi foto proyek

Proyek dengan foto menarik mendapat 3× lebih banyak klik di website. Upload foto render/site plan berkualitas tinggi.

📝

Deskripsi yang menarik

Tulis USP (Unique Selling Point) proyek di awal deskripsi: lokasi strategis, fasilitas, keunggulan dibanding kompetitor.

⚡

Respon lead dalam 5 menit pertama

Aktifkan notifikasi CRM. Begitu ada lead baru, langsung buka WhatsApp dan hubungi. Kecepatan respon = peningkatan konversi signifikan.

📡

Share link proyek secara aktif

Gunakan tombol Share WA untuk sebar link proyek ke grup WA, komunitas, dan media sosial. Semakin banyak traffic → semakin banyak leads.

📊

Pantau statistik hit secara berkala

Cek siapa agen yang paling aktif share. Koordinasi dengan agen top untuk strategi promosi bersama. Dorong agen yang pasif.

⚠️
Penting: Leads hanya masuk ke CRM Anda jika proyek berstatus Publish. Pastikan koordinasi dengan Admin untuk percepatan publikasi proyek baru.
