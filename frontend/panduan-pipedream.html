💡 01 — Konsep

Cara Kerja Integrasi

Pipedream bertindak sebagai jembatan antara Facebook Lead Ads dan CRM MANSION. Setiap prospek yang mengisi form iklan akan otomatis muncul di tab Leads CRM atas nama kamu.

📢FB/IG Lead Ads
→
🔗Pipedream

Mengapa tidak langsung dari Facebook?
Facebook Lead Ads tidak mendukung push langsung ke URL kustom. Dibutuhkan platform perantara (seperti Pipedream) yang sudah terdaftar sebagai integrasi resmi Meta.
📝 02 — Daftar

Cara Daftar Pipedream (Gratis)

1
Buka pipedream.com
Kunjungi pipedream.com dari browser. Klik tombol "Sign Up Free" di pojok kanan atas.
2
Daftar dengan Google
Pilih "Continue with Google" — paling cepat, tidak perlu isi password baru. Gunakan akun Google kerja kamu. Tidak perlu kartu kredit.
3
Kamu sudah di dashboard Pipedream
Halaman Workflows muncul. Pipedream siap digunakan. Lanjut ke langkah berikutnya.
⚙️ 03 — Setup CRM

Setup di CRM — Dapatkan Webhook URL & Secret Key

Sebelum buat workflow, siapkan dua data dari CRM: Webhook URL dan Secret Key.

1
Buka Personal Assistant
Login ke CRM. Tap ikon robot 🤖 di menu bawah untuk membuka halaman Personal Assistant (OpenClaw).
2
Buka Pengaturan PA
Tap tombol ⚙ Pengaturan PA di bagian atas. Sidebar konfigurasi terbuka dari sisi kanan.
3
Aktifkan mode Zapier
Di bagian "Webhook / Integrasi", pilih tab Zapier dan pastikan mode ini aktif (bukan "Tidak Pakai").
4
Generate Secret Key
Klik tombol "Generate Secret Key". Sebuah kode unik (UUID) akan muncul. Salin dan simpan — ini akan dipakai di Pipedream.
5
Salin Webhook URL kamu
Di bawah Secret Key, ada "Webhook URL Kamu (unik per agen)":

https://crm.mansionpro.id/api/v1/webhook/zapier/[ID_KAMU]

Tap ikon salin di sebelah URL tersebut.
Simpan keduanya sebelum lanjut:
• Webhook URL — tujuan pengiriman data lead
• Secret Key — verifikasi keamanan agar hanya workflow kamu yang bisa kirim
🔗 04 — Buat Workflow

Buat Workflow: Facebook Lead Ads → CRM MANSION

Workflow = satu alur otomasi. Kita buat alur: lead baru di FB Ads → kirim ke Webhook CRM.

BAGIAN A — Trigger: Facebook Lead Ads
1
Klik "New Workflow" di dashboard Pipedream
Tombol ada di pojok kanan atas halaman Workflows.
2
Pilih Trigger: Facebook Lead Ads
Di panel "Choose a trigger", ketik "Facebook Lead Ads". Pilih app tersebut lalu pilih event "New Lead (Instant)".
3
Connect akun Facebook
Klik "Connect Account" lalu login dengan akun Facebook yang mengelola halaman bisnis / iklan propertimu. Izinkan akses Pipedream.
4
Pilih Page & Form iklanmu
Pilih Facebook Page kamu, lalu pilih Lead Form yang mau dihubungkan. Klik "Generate Test Event" untuk mengambil contoh data lead.
BAGIAN B — Action: Kirim ke Webhook CRM
5
Tambah step baru — pilih HTTP Request
Klik tanda "+" di bawah trigger. Pilih "Send HTTP Request" atau cari app "HTTP / Webhook".
6
Pilih action: Send any HTTP Request
Pilih "Send any HTTP Request". Ini setara dengan POST di Zapier — sepenuhnya gratis di Pipedream.
7
Isi URL, Method, dan Content-Type
URL: paste Webhook URL CRM kamu
https://crm.mansionpro.id/api/v1/webhook/zapier/[ID_KAMU]

Method: POST
Content-Type: application/json
8
Isi Body JSON
Di bagian Body, pilih mode Raw. Isi JSON berikut — untuk value dinamis, klik ikon {"{}"} untuk memilih field dari data trigger:
KeyValue — pilih dari data trigger FB
secret Paste Secret Key kamu (teks statis) WAJIB
full_name Pilih Full Name dari data FB Lead WAJIB
phone Pilih Phone Number dari data FB Lead WAJIB
email Pilih Email dari data FB Lead OPSIONAL
form_name Pilih Form Name dari data FB Lead OPSIONAL
ad_name Pilih Ad Name atau Campaign Name OPSIONAL
Nama field berbeda tiap form. Gunakan variable picker Pipedream (klik ikon {}) untuk browse field yang tersedia dari data test trigger kamu.
9
Test step
Klik "Test". Jika berhasil, response dari server:
{ "received": true }
Status code harus 200. Cek juga tab Leads di CRM — data test seharusnya sudah masuk.
10
Deploy Workflow
Klik tombol "Deploy" di pojok kanan atas. Workflow sekarang aktif — setiap lead baru dari FB Ads akan otomatis dikirim ke CRM dalam hitungan detik.
📦 05 — Data

Field yang Dikirim ke CRM

Contoh lengkap payload JSON yang dikirim Pipedream ke Webhook CRM:

// Request body dari Pipedream ke CRM { "secret": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx", ← Secret Key dari CRM "full_name": "Budi Santoso", "phone": "628123456789", ← awali 62, tanpa + "email": "budi@email.com", "form_name": "Form Apartemen Jakarta Selatan", "ad_name": "Promo Akhir Tahun - Jaksel" }
Data agen ditambahkan otomatis oleh CRM:
CRM membaca agent_id langsung dari URL Webhook (sudah include ID kamu). Tidak perlu mengirim data agen dari Pipedream.
✓ 06 — Verifikasi

Test & Verifikasi Integrasi

1
Test dari Pipedream
Di editor workflow, klik "Test" pada step HTTP Request. Response {"received": true} dengan status 200 berarti CRM berhasil menerima data.
2
Cek tab Leads di CRM
Buka CRM → tab Leads. Data test seharusnya sudah muncul sebagai lead baru dengan nama kamu sebagai agen yang bertanggung jawab.
3
Test live dengan form asli
Isi form Lead Ads kamu sendiri dengan data dummy. Pipedream akan memproses lead tersebut secara real-time (biasanya dalam 5–30 detik setelah form diisi).
4
Pantau Event History
Di Pipedream → klik workflow kamu → tab "Event History". Setiap eksekusi tercatat lengkap dengan data masuk dan respons keluar.
❓ 07 — FAQ

FAQ & Troubleshooting

?
Error 401: "Invalid secret"
Secret Key di Pipedream tidak cocok dengan yang di CRM. Buka Pengaturan PA → generate ulang Secret Key → update di step HTTP Request Pipedream → Deploy ulang.
?
Error 403: "Webhook tidak aktif"
Mode webhook di CRM sedang "Tidak Pakai". Buka Pengaturan PA → Webhook/Integrasi → pilih mode Zapier → simpan.
?
Lead tidak muncul di CRM meskipun response 200
Pastikan field full_name dan phone terisi nilainya (tidak kosong). Cek di Event History Pipedream untuk melihat nilai aktual yang dikirim.
?
Tidak ada data saat "Generate Test Event"
Form Lead Ads belum pernah menerima lead, atau Pipedream belum ter-subscribe ke page tersebut. Isi form sendiri dulu, atau klik "Skip Test" dan lanjut setup — workflow tetap berjalan saat ada lead asli masuk.
?
Bisa pakai satu workflow untuk beberapa agen?
Tidak dianjurkan. Setiap agen punya Webhook URL dan Secret Key sendiri. Buat workflow terpisah untuk setiap agen / halaman Facebook yang berbeda.
?
Bisa hubungkan selain Facebook Lead Ads?
Ya. Pipedream mendukung banyak sumber: Google Forms, TikTok Lead Gen, Instagram Lead Ads, Typeform, dll. Ganti saja Trigger app-nya — Action HTTP Request-nya tetap sama.
?
Berapa limit gratis Pipedream?
Free Plan: 3.000 eksekusi/bulan (±100/hari). Artinya kamu bisa menerima hingga 3.000 lead per bulan tanpa biaya — lebih dari cukup untuk agen properti aktif.
