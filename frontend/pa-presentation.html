Anti-Bot Engine

Berperilaku Seperti Manusia Sungguhan

OpenClaw menggunakan serangkaian teknik humanisasi untuk menghindari deteksi bot oleh Instagram & WhatsApp.

🖱️

Bezier Mouse Movement

Gerakan kursor mengikuti kurva Bezier alami. Hover 80–200ms sebelum klik. Tidak pernah klik langsung/seketika.

⌨️

Human-like Typing

Mengetik 50–150ms per karakter dengan jeda acak. Simulasi "typo & hapus" sesekali. Caption IG diketik karakter per karakter.

⏱️

Randomized Delays

Jeda 3–7 detik sebelum kirim WA (simulasi mengetik). Delay 20–60 detik antar pesan. Pause 30–120 detik antar aksi besar IG.

📱

Mobile Browser Spoofing

Pool 4 user-agent iPhone/Android. Viewport 390×844 – 412×915 dengan variasi ±10px. Menyembunyikan flag navigator.webdriver.

🍪

Persistent Session

Cookie IG & WA Web disimpan di Google Cloud Storage. Reuse sesi yang sama → tidak perlu login ulang terus-menerus.

📊

Daily Limits Enforcement

Maks 5 post IG/hari. Maks 5 nomor WA/sesi · 2 sesi/hari = 10 pesan/hari. Limit dicek sebelum eksekusi.

Password Tidak Pernah Terekspos

Alur Enkripsi Kredensial
1️⃣
Agen input password IG di sidebar PA
↓ Dienkripsi AES-256-GCM sebelum dikirim
2️⃣
Tersimpan di Google Sheets sebagai ciphertext
↓ Didekripsi hanya saat job akan dieksekusi
3️⃣
Payload terenkripsi dikirim ke OpenClaw via header x-internal-secret
↓ Password TIDAK pernah masuk ke frontend
🔐
Cookie sesi disimpan terpisah di GCS Bucket

AES-256-GCM Encryption

// Enkripsi sebelum simpan ke Sheets encrypt(password): iv = random(12 bytes) tag = auth_tag(16 bytes) return "iv:tag:ciphertext" // Dekripsi hanya saat kirim ke OpenClaw decrypt(ciphertext): [iv, tag, enc] = split(":") return plaintext
